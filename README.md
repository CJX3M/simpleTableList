# Simple Table List

A modern React application featuring a responsive data table with CRUD operations, built with Vite and Bootstrap.

## Features

- **Dynamic Data Table**
  - Responsive design with horizontal scroll on mobile
  - Multi-column sorting
  - Row selection functionality
- **Inline Editing**
  - Modal-based editing form
  - Field validation
  - Real-time updates
- **Visual Feedback**
  - Animated row highlights on updates
  - Loading states
  - Error handling
- **Performance Optimizations**
  - Memoized components
  - Efficient state management
  - CSS animations

## Installation

## 1. Clone the repository:
git clone https://github.com/cjx3m/simpleTableList.git
Install dependencies:

cd simpleTableList
npm install

## Running the App
Development Mode:

npm run dev
Runs the app in development mode.
Open http://localhost:5173 to view in browser.

## Preview Build:

bash
npm run preview
Locally preview production build.
